Added the SCI (June 2026) workflow, see [gcages.sci_june_2026][].
This provides pre-processing ([gcages.sci_june_2026.pre_processing.SCIJune2026PreProcessor][]), harmonisation ([gcages.sci_june_2026.harmonisation.create_scijune2026_harmoniser][]), infilling ([gcages.sci_june_2026.infilling.create_scijune2026_infiller][]), SCM running (using [gcages.cmip7_scenariomip.scm_running.CMIP7ScenarioMIPSCMRunner][]) and post-processing (using [gcages.cmip7_scenariomip.post_processing.CMIP7ScenarioMIPPostProcessor][]).
A worked example of the full pipeline is available in the "How to run the SCI June 2026 workflow" how-to guide.
This is in line with the June 2026 SCI runs.
